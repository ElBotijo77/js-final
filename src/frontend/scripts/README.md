# Carpeta `scripts`

* **¿Qué guarda?** Scripts de JavaScript para el frontend (Vanilla JS).
  
* **Fase actual:** integración básica de los componentes para lograr el MVP.
  
* **Descripción de los módulos:**
  * `main.js`: Núcleo del juego y control del flujo principal (control de vidas).
  * `ui.js`: Gestión y actualización de la interfaz visual del usuario.
  * `questions.js` y `parseJSON.js`: Carga de preguntas desde el archivo JSON y algoritmo de barajado aleatorio de respuestas.
  * `AudioManager.js`: Control de los efectos de sonido y audio del juego.
    
* **Fase futura:** Optimizar la modularización mediante ES Modules (import/export) y asegurar la escalabilidad para futuras categorías sin tocar el backend.
